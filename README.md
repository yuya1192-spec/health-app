# 健康管理アプリ 🏥

シンプルで使いやすい健康管理PWAアプリです。体重、食事、運動、睡眠を記録して、健康的な生活をサポートします。

## 主な機能

### 📊 ダッシュボード
- 体重、BMI、カロリー収支、睡眠時間を一目で確認
- 30日間の体重推移グラフ
- 7日間のカロリー摂取・消費グラフ
- 7日間の睡眠時間グラフ

### ⚖️ 体重管理
- 体重の記録と履歴管理
- BMI自動計算
- 体重推移の可視化

### 🍽️ 食事管理
- 食事とカロリーの記録
- 目標カロリー設定
- よく食べる食事のクイック登録
- 日別の摂取カロリー集計

### 🏃 運動管理
- 運動の種類と時間の記録
- 消費カロリーの計算
- 運動履歴の管理

### 😴 睡眠管理
- 就寝・起床時刻の記録
- 睡眠時間の自動計算
- 睡眠の質の評価
- 平均睡眠時間の統計

## 技術スタック

- **HTML5** - セマンティックなマークアップ
- **CSS3** - モダンなレスポンシブデザイン
- **JavaScript (ES6+)** - バニラJSで実装
- **Chart.js** - 美しいグラフ表示
- **PWA** - オフライン対応、ホーム画面追加可能
- **LocalStorage** - データの永続化

## 使い方

### デスクトップ
1. `index.html` をブラウザで開く
2. 各タブから記録を追加

### スマートフォン
1. ブラウザでアプリにアクセス
2. 共有ボタンから「ホーム画面に追加」
3. アプリアイコンから起動

## インストール

```bash
# リポジトリをクローン
git clone https://github.com/あなたのユーザー名/health-app.git

# ディレクトリに移動
cd health-app

# ブラウザで開く
# index.html をダブルクリック、または
# ローカルサーバーを起動
python -m http.server 8000
# http://localhost:8000 にアクセス
```

## ブラウザ対応

- ✅ Chrome / Edge (推奨)
- ✅ Safari (iOS対応)
- ✅ Firefox
- ✅ モバイルブラウザ

## データについて

- すべてのデータはブラウザのLocalStorageに保存されます
- データはデバイス内に保存され、外部に送信されません
- ブラウザのキャッシュをクリアするとデータが削除されます
